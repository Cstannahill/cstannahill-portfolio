CodeShift is a modern, AI-powered platform designed to help developers translate code across frameworks like React, Vue, Next.js, and Angular—while simultaneously building personalized learning paths to deepen their understanding of each technology. It blends practical tooling with smart insights to create a full development and education ecosystem.

## 🧩 Core Challenges

<ProjectChallengeCard
  title="Framework-Aware Code Translation"
  challenge="Traditional converters fail to respect idioms like hooks, routing, or composition models across JS frameworks."
  solution="Implemented a hybrid LLM + static analysis approach to preserve structure and map functional intent."
  impact="Dramatically improved translation accuracy and usability in real-world projects."
  difficulty="high"
  domain="AI / Dev Tooling"
/>

<ProjectChallengeCard
  title="Adaptive Learning at Scale"
  challenge="Pre-built tutorials often mismatch developer level, causing disengagement or inefficiency."
  solution="Designed adaptive curricula based on analysis of past projects, target goals, and usage patterns."
  impact="Improved engagement and knowledge retention while reducing redundant lessons."
  difficulty="medium"
  domain="Learning Platforms"
/>
